# FedoraDev
A shell script to have a baller Fedora development environment

## Download
```shell
git clone https://github.com/DrewImm/FedoraDev.git
```

## Install
```shell
cd FedoraDev && chmod 755 dev-install.sh
./dev-install.sh
```
